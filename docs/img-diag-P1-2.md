## 0. 한 줄 결론

**핵심 병목(카피-이미지 정합성)이 해소됐다.** 공감=붉어진 피부, 원인=피부 접촉, CTA=제품 히어로로 **역할이 실제로 갈렸다**. P1-1에서 전부 "제품 정물"로 수렴하던 문제가 image_mission 우선 설계로 해결됨.

## 1. V2가 만든 image_mission (코드 가드 작동 확인)

| 역할 | product_visibility(코드 clamp) | visual_focus | shot_type(결과값) |
|---|---|---|---|
| ①공감 | **10%** (band 0~20) | 망설이는 손끝 + 살짝 붉어진 볼·턱선 피부 | 감정 클로즈업(피부·손) |
| ②원인 | **0%** (band 0~20) | 당긴 턱선 피부를 누르는 순간(자극 누적의 신체감각) | 원인 각성 클로즈업(피부 접촉) |
| ③솔루션 | **55%** (band 40~70) | 손등에 닿는 투명 토너 한 방울 + 제품 + 병풀 | 솔루션 플랫레이+손 |
| ④신뢰 | **70%** (band 50~80) | 클리니컬 배경 제품 + 손바닥 위 투명 제형 | 클리니컬 정물+신뢰 클로즈업 |
| ⑤CTA | **95%** (band 80~100) | 제품 히어로 + 황금빛 사이드 조명 | 히어로 제품컷 |

→ shot_type이 **출발점이 아니라 image_mission의 결과**로 도출됨. product_visibility는 섹션 role 기반 코드 clamp가 정확히 적용.

## 2. P1-1 평가표 재검증

| 항목 | P1-1 (구버전) | **P1-2 (V2)** |
|---|---|---|
| **① 카피-이미지 정합성** | △~✕ 전부 제품 정물, 내용 불일치 | **◯ 해결** — 공감=붉은 볼(카피 '볼이 빨개져 당황'과 일치), 원인=자극받는 피부, 솔루션=진정, CTA=초대 |
| **② 섹션 역할 분화** | △ 구도만 다름, 전부 제품 | **◯◯ 극적 개선** — 공감(피부)/원인(접촉)/솔루션(제품+손)/신뢰(클리니컬)/CTA(히어로) 내용 차원에서 갈림 |
| **③ 제품 일관성** | B에서만 ◯ | **◯ 유지** — 제품 등장 컷(③④⑤) 모두 동일 ACUA 센텔라 칼밍 토너(레퍼런스 유지) |
| **④ 텍스처 정확성** | B에서 ◯ | **◯ 유지** — 투명 워터리, 레퍼런스 따라 보수적 |
| **⑤ 성분 정확성** | ◯ | **◯** — 병풀 정확, 다른 원료 오인 없음 |
| **⑥ 블로그형 적합성** | ◯ | **◯** — 텍스트 오버레이 0, 광고배너 아님, 가짜 인증/수치 없음 |

### ⭐ 핵심 합격 기준(작업 4번)
- **공감 = 제품컷 아니라 "붉어진 피부"로 갈렸나? → YES.** 붉어진 볼·턱선 클로즈업 + 만지는 손, 제품은 배경에 흐릿(10%).
- **CTA = "히어로 제품"으로 갈렸나? → YES.** 골든 사이드라이트 제품 히어로(95%).

### 인물 정책 준수
공감/원인 이미지는 **식별 불가한 얼굴 일부(코·입·턱선)·손·피부**만 — 눈/전체 얼굴 없음. "동일 모델 얼굴·화보 금지 / 신체 일부 허용" 정책대로.
