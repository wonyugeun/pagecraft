## 2. 핵심 진단 — 병목은 브리프도 모델도 아니다. **레퍼런스(제품 일관성)다.**

### ✅ 모델(Gemini) 품질: 매우 우수
- 4컷 모두 프리미엄 K-beauty 에디토리얼 룩. 조명·여백·질감 표현 충분히 상품화 가능 수준.
- 블로그형 텍스트 금지 규칙이 실제로 먹힘(카피·헤드라인·숫자 오버레이 없음).
- 단일 컷 미감만 보면 **추가 작업 불필요할 만큼 좋다.**

### ✅ 브리프(Stage4) 품질: 단일 컷 단위로는 충분
- shot_type이 정확히 분기됨: 단독컷=제품 중심, 성분컷=flat-lay+병풀, 텍스처=워터리 매크로, 사용=손/피부 클로즈업(얼굴 없음).
- mood/palette도 전략 톤(차분·워터리·에디토리얼)을 일관 번역.

### ⚠️ 진짜 병목: **제품 정체성이 컷마다 다르다**
레퍼런스 이미지(productImages)가 없어서 Gemini가 매 컷 **다른 브랜드·다른 병**을 발명:

| 컷 | 발명된 브랜드/라벨 | 병 형태 |
|---|---|---|
| 1 단독 | **ACUA** / 센텔라 CALMING TONER 150ml | 반투명 실린더 |
| 2 성분 | **PURITO SEOUL** / Centella Unscented 200ml | 녹색뚜껑 투명 |
| 3 텍스처 | **AQUA CALM TONER** / "K-BEAUTY EDITORIAL" 200ml | 각진 투명 |
| 4 사용 | **MUZAKO** / Clear Soothing Toner | 흰뚜껑 투명 |

→ 상세페이지는 같은 제품이 위→아래로 반복돼야 하는데, **컷마다 다른 제품처럼 보인다.** 이게 페이지 단위 최대 결함.
→ 원인: `generate-image` route의 `PRODUCT_RULES`/`COMPONENT_RULES`는 `hasRefImages`일 때만 활성. 레퍼런스 없으면 일관성 규칙이 사실상 작동 안 함.

### (부차) 발명된 라벨 텍스트
일부 병에 "K-BEAUTY EDITORIAL" 등 가상 라벨 문구가 새겨짐. 오버레이가 아니라 "제품 자체 라벨"로 들어가 블로그 텍스트 금지에는 안 걸리지만, 레퍼런스 있으면 자연 해소될 항목.

## 3. 결론 / 다음 우선순위

- **"브리프 심화는 시기상조"가 데이터로 확인됨.** 단일 컷 미감은 모델·브리프 모두 이미 충분.
- **블로그형 이미지 품질의 1순위 과제 = 제품 레퍼런스 배선** (productImages 주입 → 모든 컷 동일 제품 유지). 브리프 분기 심화보다 이게 먼저.
- 슬라이드형(광고형)은 이번 범위 밖. 블로그형 기준만 확인.

## 4. 과금 / 소요

- Gemini 이미지 생성 **4건** (`gemini-3.1-flash-image-preview`) — 진단용 1회, 재시도 없이 4건 모두 1회 성공.
- Claude 브리프 호출 **1건** (`claude-sonnet-4-6`, 1청크, max_tokens 8000).
- 총 소요 **73.9초** (브리프 + 이미지 4컷 순차).
- 정확한 원화 과금액은 단가표 미확인 — 호출 단위는 위와 같음(이미지 4 + 텍스트 1).
